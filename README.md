# Emotion-Classifier from Voice using Whisper & CNN

![Python](https://img.shields.io/badge/Python-3.9-blue.svg)
![PyTorch](https://img.shields.io/badge/PyTorch-1.13+-ee4c2c.svg)
![Whisper](https://img.shields.io/badge/Whisper-OpenAI-blueviolet)

This project performs **emotion classification (Angry, Sad, Neutral, Happy)** using audio samples. It leverages **OpenAI's Whisper** model to extract `log-mel spectrograms`, which are then fed into a **Convolutional Neural Network (CNN)** for classification.
